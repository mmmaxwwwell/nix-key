Removed `.trim()` before validation in `connectWithAuthKey()` so that `isValidAuthKeyFormat()` receives the raw user input including any whitespace, which the existing whitespace regex check correctly rejects. The trimmed key is now computed separately after validation passes and used only for the actual `tailscaleManager.start()` call. Additionally, added inline validation in `onAuthKeyChanged()` following the `SettingsViewModel.setOtelEndpoint()` pattern — the auth key is validated on every keystroke and the error state is updated immediately, providing instant feedback when the user enters whitespace or other invalid characters.
